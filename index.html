Getting started

💰Cost: ~$0.15/video

⚠️ Do not use if free tier, text-to-speech limit is 3 per minute. ⚠️

  1. 🧠 Add OpenAI API Key
    • Only sent to OpenAI
  2. 🔉 Turn on sound
  3. ✍️ Enter what you want to learn
  4. 🎥 Hit "New Video" button
  5. 🎓 Start learning
  6. 📠 Save as a video and share
